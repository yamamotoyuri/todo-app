# 基本設計書

## 📋 プロジェクト概要

このプロジェクトは、**タスク管理アプリケーション**です。ユーザーはタスクを作成、管理、削除することができます。

---

## 🛠 使用技術

| 分類               | 技術         |
| ------------------ | ------------ |
| **バックエンド**   | PHP, Laravel |
| **フロントエンド** | React, Vite  |
| **データベース**   | MySQL        |

---

## 🚀 環境構築

### セットアップ手順

1. **プロジェクトディレクトリへ移動**

   ```bash
   cd C:\Users\yuris\todo-app
   ```

2. **Dockerコンテナを起動**

   ```bash
   docker compose up -d
   ```

3. **バックエンドサーバーを起動** (ターミナル1)

   ```bash
   docker compose exec backend php artisan serve --host=0.0.0.0 --port=8000
   ```

   - 📍 バックエンド: `http://localhost:8000`

4. **フロントエンドサーバーを起動** (ターミナル2)

   ```bash
   docker compose exec frontend npm run dev
   ```

   - 📍 フロントエンド: `http://localhost:5173`

## 📊 データベース

### テーブル一覧

#### 📌 tasks テーブル

タスク情報を管理するメインテーブル

| カラム名     | 型          | 説明                            |
| ------------ | ----------- | ------------------------------- |
| id           | bigint      | 主キー（自動採番）              |
| title        | string(255) | タスクのタイトル                |
| is_completed | boolean     | 完了フラグ（デフォルト: false） |
| created_at   | timestamp   | 作成日時                        |
| updated_at   | timestamp   | 更新日時                        |

#### 📌 sessions テーブル

セッション情報を管理するテーブル

| カラム名      | 型         | 説明                                   |
| ------------- | ---------- | -------------------------------------- |
| id            | string     | セッションID（主キー）                 |
| user_id       | bigint     | ユーザーID（null許可）                 |
| ip_address    | string(45) | IPアドレス                             |
| user_agent    | text       | ユーザーエージェント                   |
| payload       | longtext   | セッションデータ                       |
| last_activity | integer    | 最終アクティビティのUNIXタイムスタンプ |

#### 📌 personal_access_tokens テーブル

APIアクセストークンを管理するテーブル

| カラム名       | 型         | 説明                   |
| -------------- | ---------- | ---------------------- |
| id             | bigint     | 主キー（自動採番）     |
| tokenable_type | string     | トークン所有者の型     |
| tokenable_id   | bigint     | トークン所有者のID     |
| name           | text       | トークン名             |
| token          | string(64) | トークン値（ユニーク） |
| abilities      | text       | トークンの権限         |
| last_used_at   | timestamp  | 最終使用日時           |
| expires_at     | timestamp  | 有効期限               |
| created_at     | timestamp  | 作成日時               |
| updated_at     | timestamp  | 更新日時               |

## ✨ 主要機能

- タスクの作成、更新、削除
- タスクの状態管理

---

## 📝 機能一覧

| 機能名             | 説明                              | 状態        |
| ------------------ | --------------------------------- | ----------- |
| タスク作成         | 新しいタスクの追加                | ✅ 実装済み |
| タスク一覧表示     | 全タスクを一覧表示                | ✅ 実装済み |
| タスク編集         | 既存タスクの更新                  | ✅ 実装済み |
| タスク削除         | タスクの削除                      | ✅ 実装済み |
| タスク状態管理     | タスクの完了/未完了状態を切り替え | ✅ 実装済み |
| データエクスポート | タスクデータのエクスポート        | ✅ 実装済み |
